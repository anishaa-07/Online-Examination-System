# Online-Examination-System
A Java-based online examination system with login, MCQs, timer, and auto-submit features.

An Online Examination System developed using Core Java that simulates a real-world exam environment. The application allows users to securely log in, update their profile and password, attempt multiple-choice questions, and submit the exam automatically when the timer ends. The system also supports proper session handling and logout functionality.
